# 包描述文件 
import { Tag } from "../../components/tag.tsx"
 
描述 Nep 包的基本信息，位于 [`package.toml`](/nep/struct/2-inner.html#包描述文件) 
## package 
通用信息表
 
### name
包名
* 类型：`String`  
* 示例：`name = "VSCode"`

### description
包的简短描述，尽量从官方渠道摘取
* 类型：`String`  
* 示例：`description = "微软出品的开源编辑器"`

### template
包模板，当前版本中仅能为 "Software"
* 类型：`String`  
* 示例：`template = "Software"`

### version
包版本号，使用 ExSemVer 规范
* 类型：`String`  
* 示例：`version = "1.0.0.0"`

### authors
包作者，第一作者应为打包者，后面通常跟发行商、制作方

支持使用 `<>` 包裹作者邮箱
* 类型：`Vec<String>`  
* 示例：`authors = ["Cno <dsyourshy@qq.com>", "Microsoft"]`

### license
<Tag>可选</Tag> 开源许可证的 [SPDX 标识符](https://spdx.org/licenses/)或 EULA 链接
* 类型：`String`  
* 示例：`license = "MIT"`

### icon
<Tag>可选</Tag> 包图标 URL
* 类型：`String`  
* 示例：`icon = "https://code.visualstudio.com/favicon.ico"`
