## 内置变量

### SystemDrive
系统盘符 
* 示例值：`C:` 
* 示例：`if = 'SystemDrive=="C:"'`
### Home
用户主目录 
* 示例值：`C:/Users/UserName` 
* 示例：`if = 'Home=="C:/Users/UserName"'`
### AppData
应用数据目录

:::warning
注意：此变量和 Windows 系统内的 `%appdata%` 变量表示的目录位置不同，其不会自动在末尾加上`Roaming`
::: 
* 示例值：`C:/Users/UserName/AppData` 
* 示例：`if = 'AppData=="C:/Users/UserName/AppData"'`
### ProgramFiles_X64
ProgramFiles 目录 
* 示例值：`C:/Program Files` 
* 示例：`if = 'ProgramFiles_X64=="C:/Program Files"'`
### ProgramFiles_X86
ProgramFiles (x86) 目录 
* 示例值：`C:/Program Files (x86)` 
* 示例：`if = 'ProgramFiles_X86=="C:/Program Files (x86)"'`
### Desktop
当前用户桌面目录 
* 示例值：`C:/Users/UserName/Desktop` 
* 示例：`if = 'Desktop=="C:/Users/UserName/Desktop"'`
### PublicDesktop
公用桌面目录 
* 示例值：`C:/Users/Public/Desktop` 
* 示例：`if = 'PublicDesktop=="C:/Users/Public/Desktop"'`
### Arch

* 示例值：`X64` 
* 示例：`if = 'Arch=="X64"'`
